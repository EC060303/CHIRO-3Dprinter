# CHIRO-3Dprinter
이 repo는 중앙대학교 통일공대 로봇 동아리 CHIRO를 위한 3D 프린터 사용 가이드 및 정비 가이드 관련 문서입니다.

# 동아리방 프린터 목록
동아리방에는 총 3대의 3D 프린터가 있으며 종류는 아래와 같습니다.

* Voron 2.4 (수리 중)
* Sapphire Plus v3 trident (수리 중)
* Bambu Lab P1S

각 프린터의 사진은 위 소제목을 클릭하면 확인하실 수 있습니다.

### ⚠️ 사용 전 사용 내역을 작성해 주시기 바랍니다.
### 내역서 작성 없이 출력 중인 경우, 출력이 취소될 수 있습니다.

# [Bambu Studio 사용법](./BambuStudio)
Bambu Studio는 3D 프린팅을 위해 사용되는 슬라이서로, 위 제목을 클릭하면 설치법 및 상세 사용법을 확인할 수 있습니다.

# [3D 프린터 출력 방법](./3Dprinting)
3D 프린터를 이용하여 출력물을 출력하는 방법입니다. 위 제목을 클릭하면 상세 내용이 나옵니다.

# 출력 이후 출력물 제거 방법
### ⚠️ 절대로 금속 헤라나 칼을 이용해서 제거하지 마세요.
### PEI시트가 손상됩니다.
### 출력직후 바로 제거하실 경우 뜨거우니 장갑을 끼시고 제거하시거나 식으면 제거하세요.

# [필라멘트 교체 방법](./Filament_Change)
필라멘트 교체 방법입니다. 위 제목을 클릭하시면 상세 내용이 나옵니다.
