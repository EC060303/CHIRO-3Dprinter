Bambu Studio를 이용하여 출력하기 위해서는 우선 노트북이 저희 동아리 와이파이를 연결해야합니다.
* 와이파이 이름: CHIRO
* 와이파이 비밀번호: chiro2001

![프린터연결1](./images/Printing1.png)

와이파이 연결 후 위 Bambu Studio에서 장치 화면에 들어갑니다.

![프린터연결2](./images/Printing2.png)

이후 액세스 코드 바인딩을 눌러서 동아리방 프린터와 연결하면 됩니다.
* Bambu Lab P1S ip: 192.168.0.73
* Bambu Lab P1S 액서스 코드: c0e26059

프린터기와 연결한 이후 프린터를 동기화 해주어야합니다.

![프린터연결3](./images/Printing3.png)

노즐 정보 동기화 이후 필라멘트 동기화도 진행해주시면 됩니다.

![프린터연결4](./images/Printing4.png)
![프린터연결5](./images/Printing5.png)
